Вы работаете в компании, занимающейся управлением сетью зарядных станций для электромобилей. В вашем распоряжении имеются сведения о владельцах электромобилей, включающие стандартные анкетные данные (фамилия, имя, отчество, адрес, телефон). Ваша компания должна обеспечить функционирование зарядных станций в различных локациях города. По каждой зарядной станции установлены определенные характеристики: количество зарядных портов, типы разъемов, максимальная мощность зарядки. В результате работы с клиентами у вас появляется информация о том, какие станции использует каждый владелец электромобиля. Существует некоторый минимальный уровень использования сети, который должен поддерживаться для обеспечения рентабельности. По окончании каждого месяца вы заносите информацию о количестве зарядных сессий и потребленной электроэнергии для каждого клиента.
